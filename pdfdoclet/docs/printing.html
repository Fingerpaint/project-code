Printing

Basic

If the PDF is to be used for printing mainly, there are a few configuration properties to consider. First of all, make sure that printing is enabled: 

        encrypted=yes
        allow.printing=yes

Custom Page Breaks

If the document is to be printed well, it may be necessary to insert custom page breaks in the overview.html or some package.html files. A custom page break can be triggered with a special HTML comment: 

      <!-- NEWPAGE -->

Disabling Hyperlinks

In a printed document, links are pretty much useless, so it may be better to disable them (to make them be printed black and not underlined): 

        create.links=no
